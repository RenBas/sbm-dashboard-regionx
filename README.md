## 📌 Overview

Interactive dashboard for monitoring School-Based Management (SBM) implementation across DepEd Region X (Northern Mindanao). Built with **Streamlit** and **Folium** to visualize SBM performance across 14 Schools Division Offices (SDOs) and their constituent schools.

### Key Features
- 🗺️ **Interactive Map** with color-coded SDO shields (pulse/glow indicates urgency)
- 🏫 **School-level visualization** with color-coded dots based on SBM degree
- 📊 **KPI Cards** showing division-level performance metrics
- 📋 **Indicators Table** with 42 SBM indicators
- 📈 **Radar Chart** comparing division vs regional averages
- 🔍 **Search** functionality to find specific schools
- 🧪 **Digital Twin Sandbox** (coming soon) for policy simulation
